Sobre a VivaDrive

A VivaDrive Locadora é uma empresa fictícia criada como parte de um projeto escolar para demonstrar habilidades em desenvolvimento web e design de interfaces.

Nosso objetivo com este projeto é simular um sistema de locação de veículos completo, com funcionalidades como cálculo de preços, seleção de veículos e formulários de cadastro.

Este é um Trabalho Escolar

Disciplina: FTP

Objetivo: Criar um site funcional para locação de veículos

Recursos implementados: Formulários interativos, cálculo dinâmico de preços, design responsivo

Observação: Todos os dados e imagens são fictícios, utilizados apenas para fins educacionais.
